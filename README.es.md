## ¿Qué es DandaraAI?

DandaraAI es un ecosistema de asistente de IA para desktop y mobile creado por SkylineTech. Conecta lenguaje natural, WhatsApp, CRM, documentos, calendario, archivos, control local del computador, flujos técnicos y mini apps personalizadas en una sola asistente capaz de entender solicitudes y ejecutar trabajo.

El módulo de ventas es importante, pero Dandara no se limita a ventas: está diseñada para ayudar a personas, pequeños negocios y equipos a operar su día a día desde chat, voz, WhatsApp o escritorio.

## Capacidades principales

- **Chat y voz** con contexto entre canales.
- **WhatsApp AI** para responder clientes, hacer seguimiento y enviar archivos.
- **CRM y memoria de clientes** para contactos, conversaciones y oportunidades.
- **Documentos y Office** para propuestas, reportes, facturas, PDFs, hojas de cálculo y emails.
- **Automatización de escritorio** para abrir apps, buscar archivos, hacer capturas y operar herramientas locales con permisos explícitos.
- **Ventas sociales** con catálogo, agente IA, Instagram/Facebook/WhatsApp, posts programados y traspaso a humano.
- **Trabajo técnico** para código, issues de GitHub, resúmenes de sprint y scripts.
- **Integraciones** con herramientas como Gmail, Calendar, Drive, Slack, Notion, Trello, GitHub, HubSpot, Jira y Stripe.
- **Mini apps** para inventario, finanzas, dashboards, onboarding y flujos internos.
- **Seguridad** con enfoque local-first, sandbox, auditoría, políticas de riesgo y confirmación humana para acciones sensibles.
